# The Solution

## Arc Treasury: T-Bill yield in one click

| Feature | Description |
|---------|-------------|
| **Treasury Vault** | Deposit USDC → earn ~4% APY from T-Bills via USYC |
| **Cross-chain Bridge** | Ethereum ↔ Arc via Circle CCTP V2 |
